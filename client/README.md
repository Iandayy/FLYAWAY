# 🖥 client 개발 기술 상세

<br />

## 📋 프로젝트 구조

- 회원 가입 (join)

- 로그인 (login)
  - jwt 로그인 방식

- 로그아웃 (logout)

- 마이페이지 (mypage)
  - 요약
  - 캘린더
  - 프로필 편집
  - 회원 탈퇴

- 운동하기 게시판 (videos)
  - 조회수 Top 10
  - 홈트레이닝
  - 스트레칭

- 공유하기 게시판 (posts)
  - 전체
  - 운동 기록
  - 다이어트 식단
  - 자유

<br />

## 🗓 개발 일정

1.  Week 1 (09/14 ~ 09/20)
- 화면 골격 만들기

- 운동하기 게시판

- 로그아웃 기능
